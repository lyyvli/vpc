# CreateCommonBandwidthPackage {#doc_api_Vpc_CreateCommonBandwidthPackage .reference}

调用CreateCommonBandwidthPackage接口创建共享带宽实例。

请确保在使用该接口前，已充分了解该产品的[计费](~~89732~~)。
