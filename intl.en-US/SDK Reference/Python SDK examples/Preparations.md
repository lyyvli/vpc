# Preparations {#concept_wtb_2wy_3hb .concept}

Before you run example VPC Python SDK files, you must complete the required configurations and ensure the following:

1.  There is at least 14.88 USD in your account.
2.  You have obtained an AccessKey. For more information, see [Create an AccessKey](../../../../reseller.en-US/General Reference/Create an AccessKey.md#).
3.  The [VPC Python Example library](https://github.com/aliyun/aliyun-openapi-python-sdk-examples) is downloaded.
4.  The setup.py directory is accessed and the following command is run to initiate the environment.

    ```
    python setup.py install --user
    ```
